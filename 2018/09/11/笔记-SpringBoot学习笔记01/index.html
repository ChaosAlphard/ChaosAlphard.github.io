Spring学习笔记

环境配置:
JDK: Java 8
IDE: Intellij IDEA
服务器: Tomcat 9

在IDEA中创建Spring项目

  • Create Project -> Spring Initializr
    1. 选择JDK
    2. 选择Initializr Service URL (https://start.spring.io)
    3. 选择插件
      • Core -> DevTools
      • Web -> web
      • Template Engines -> Thymeleaf
    4. 完成

创建Controller来控制地址映射

  1. 创建容纳Controller的包和类
  2. 添加注解@Controller
  3. 添加路径映射@RequestMapping

  2. 创建模板

  3. 启动服务器
