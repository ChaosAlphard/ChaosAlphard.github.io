mysqldump常用选项

  • –databases
    指定需要备份的数据库名称
    用法:

  • –all-databases
    备份所有的数据库
    用法:

  • –tab[=dir_name]
    　　不使用此选项，mysqldump仅输出一个sql文件。
    　　使用此选项，mysqldump会对每一个要备份的表输出两个文件，sql文件保存创建表时的sql语句，txt文件储存表中的数据。
    　　dirname即为sql和txt文件输出的路径。

  • –add-drop-table
    　　此选项会在每一个表前面加上drop table if exists 语句。

  • –add-locks
    　　此选项会在insert语句前面加上lock table 语句。

  • –skip-comments
    　　此选项会去掉输出文件中的注释
